递归就是函数调用自己：伪代码如下

```go
func countdown(box int) {
	if box == 0 {
		return
	}
	println(box)
	countdown(box - 1)
}
```

编写递归函数时，必须要告诉它何时停止递归，所以每个递归函数都有两部分：基线条件(base case)和递归条件(recursive case) 递归条件指函数调用自己，基线条件则指函数不再调用自己，从而避免形成无限循环。
